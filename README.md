fex 
==========================
A frontend framework with gulp and browserify.   

## Features
- 组件化和支持子组件
- 使用eslint做js代码检查(es5)
- livereload
- 使用postcss
- 使用Modernizr
- 支持webp

## Requires
    nodejs >=v5.8.0

## Install
    npm install    

## Usage

run dev server: 
```console
npm run dev
```

run release server: 
```console
npm run release  
```

bulid: 
```console
npm run build component
```

clean: 
```console
npm run clean component
```
watch：
```console
npm run watch component
```
